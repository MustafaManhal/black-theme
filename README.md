# Black Theme

A collection of pure black VS Code themes with visible borders and 8 carefully crafted variants — from minimal to neon.

## Variants

| Theme | Background | Style |
|---|---|---|
| **Black Bordered** | `#000000` | The original — clean borders, balanced syntax |
| **Black Borderless** | `#000000` | Original Black Bordered styling with no visible UI borders |
| **Black Bordered Lite** | `#141414` | Softer background, slightly warmer feel |
| **Black Bordered High Contrast** | `#000000` | Stronger borders, brighter syntax for maximum readability |
| **Black Bordered Warm** | `#0A0805` | Warm tint with amber/gold/peach syntax |
| **Black Night** | `#060810` | Deep navy-black with cool blue syntax throughout |
| **Black Mocha** | `#0C0806` | Dark brown-black with earthy amber and rust tones |
| **Black Neon** | `#000000` | Pure black with vibrant neon green, pink, and cyan syntax |
| **Black Minimal** | `#000000` | Pure black, fully desaturated — ultra clean and distraction-free |

## Installation

1. Open VS Code
2. Press `Ctrl+Shift+P` (or `Cmd+Shift+P` on Mac)
3. Type `Preferences: Color Theme`
4. Search for **Black** and pick your variant

## Features

- Visible borders on every panel, sidebar, tab bar, status bar, and terminal
- Consistent border color per variant — no random transparent gaps
- Full syntax coverage: JavaScript, TypeScript, React/JSX, Python, Go, CSS, Markdown
- Semantic token support for accurate highlighting in TS/JS
