# Changelog

## [0.1.0] — 2026-06-02

### Added
- **8 theme variants:** Black Bordered, Black Bordered Lite, Black Bordered High Contrast, Black Bordered Warm, Black Night, Black Mocha, Black Neon, Black Minimal
- Full semantic token support across HTML, CSS, JS/JSX, JSON, Markdown, Shell, and Rust
- **README** with variant overview, installation guide, and feature list
- Visible borders on all panels, sidebars, tab bars, terminals, and widgets
- Distinct syntax highlighting for HTML/CSS/JSX and SCSS/SASS structures
- Regex, shell, and Rust syntax enhancements for better readability

### Changed
- Extension renamed to **Black Theme**
- Version bumped to `0.1.0`
- Improved border consistency across all variants
- Updated syntax colors so HTML tags, attributes, CSS selectors, values, units, properties, variables, and at-rules are readable and consistent
- Markdown headings and JSON punctuation now have clearer token styling
- Variables now use a deep, noticeable blue for easier scanning
- Each variant palette now matches its tint direction: Night = blues, Mocha = ambers, Neon = saturated brights, Minimal = desaturated muted blacks
